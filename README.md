# Portfolio Web Profesional

Portfolio profesional desarrollado para presentar mi perfil como Desarrollador de Aplicaciones Multiplataformas.

## Tecnologías

- React
- Vite
- JavaScript
- CSS
- React Icons

## Características

- Diseño responsive
- Arquitectura basada en componentes reutilizables
- Secciones organizadas por componentes
- Optimización para producción
- SEO básico
- Open Graph
- Accesibilidad básica

## Secciones

- Inicio
- Sobre mí
- Proyectos
- Tecnologías
- Experiencia profesional
- Contacto

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/jorgegrhlm/portfolio-web.git
```

Instalar dependencias:

```bash
npm install
```

Ejecutar en desarrollo:

```bash
npm run dev
```

Generar la versión de producción:

```bash
npm run build
```
